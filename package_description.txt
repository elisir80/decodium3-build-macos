Decodium is a weak-signal amateur radio application fork maintained by
9H1SR for macOS and Linux. It is based on Decodium 3 / WSJT-X lineage
and is packaged in this repository for Apple Silicon, Intel macOS, and
Linux AppImage users.
.
The fork focuses on practical operating improvements around FT2 while
continuing to support FT4, FT8, JT4, JT9, JT65, Q65, FST4, MSK144,
WSPR, FST4W, and Echo. Recent release work includes FT2 anti-ghost
filtering for very weak decodes, Costas sync refinements for FT2/FT4/
FT8, web remote-console parity controls, complete web-app localization
that follows the desktop UI language, secure settings hardening, and
safer network download handling.
.
Decodium provides wideband spectral displays, CAT control for modern
radios, online integrations such as PSK Reporter and DX services, and
release packaging tuned for macOS and Linux users. The repository also
contains multilingual release notes and startup guidance for macOS
quarantine handling and Linux AppImage execution.
.
Be sure to read the release notes and documentation shipped with the
fork before deploying it in station use.
